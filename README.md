## Функціонал

- Вкладки (Tabs) для розділення інформації за категоріями
- Акордеони (Accordion) для зручного перегляду та приховування контенту
- Анімації кнопок та зображень
- Адаптивний дизайн для екранів 900px і 500px

## Використані технології

- React
- JavaScript (ES6)
- CSS
- React Icons
- Компоненти Tabs та Accordion для організації контенту
